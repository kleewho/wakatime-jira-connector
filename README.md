# wakatime-jira-connector

Downloads your activities from wakatime and reports times as worklog on jira

## Installation

Download from https://github.com/kleewho/wakatime-jira-connector.

## Usage


    $ java -jar wakatime-jira-connector-0.1.0-standalone.jar [args]
